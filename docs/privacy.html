Privacy Policy

Last updated: March 23, 2026

Overview

Sunshine is a browser extension that overlays decorative shadow effects on websites. It is designed with privacy as a core principle.

Data Collection

Sunshine does not collect, transmit, or share any data. Specifically:

  • No personal information is collected.
  • No browsing history or activity is tracked.
  • No analytics or telemetry data is gathered.
  • No data is sent to any server or third party.

Local Storage

Sunshine stores your preferences (which shadow style and opacity you've chosen for each website) locally on your device using the browser's built-in storage.local API. This data:

  • Never leaves your device.
  • Is only accessible to the Sunshine extension.
  • Is automatically removed if you uninstall the extension.

Permissions

Sunshine requests the following permissions:

  • Access to all websites — required so the shadow overlay can appear on any page you visit. No page content is read or modified beyond adding the visual overlay.
  • Storage — used to save your per-site preferences locally.
  • Active Tab — used to communicate between the popup and the current page when you interact with Sunshine's controls.

Third-Party Services

Sunshine does not integrate with or send data to any third-party services.

Changes to This Policy

If this policy changes, the updated version will be published at this URL. Since Sunshine collects no data, significant changes are unlikely.

Contact

For questions about this privacy policy, please open an issue on the Sunshine GitHub repository.
